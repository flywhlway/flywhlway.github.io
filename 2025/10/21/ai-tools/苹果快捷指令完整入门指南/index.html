1. 什么是”快捷指令”

“快捷指令”（Shortcuts）是苹果官方提供的自动化工具，原生预装于 iPhone、iPad、Mac、Apple Watch 和 Vision Pro。它允许你把多个”动作”（Actions）串联起来，通过一次点击、一句 Siri 指令、或某个事件触发，自动完成一连串原本需要手动操作的任务。

核心理念：不会编程，也能自动化。你可以把它理解为”可视化的低代码工作流编辑器”，类似 Windows 上的批处理脚本或 Automator，但更现代、更跨平台、更智能。

在 iOS 26 / macOS 26 中，快捷指令获得了多年来最大规模的更新——首次接入 Apple Intelligence 与 ChatGPT，Mac 端首次支持完整的个人自动化，并与 Spotlight、控制中心、菜单栏深度整合。

2. 产品特性总览（最新版本亮点）

2.1 共享亮点（iOS 26 & macOS 26）

  • 接入 Apple Intelligence（关键新功能）：新增 “Use Model” 动作，可直接调用 Apple 设备端模型或 ChatGPT，输出结果可作为变量喂给后续步骤。
  • 写作工具套件（Writing Tools）：内置 6 个 AI 文本动作——校对、改写、调整语气、总结、生成列表、生成表格。
  • 图像智能（Image Playground & Visual Intelligence）：通过快捷指令直接生成 AI 图片，或调用视觉智能识别屏幕/照片内容。
  • 25+ 全新动作：覆盖 Messages、Photos、Stocks、Sports、Reminders、Voice Memos、Screen Time、Weather 等系统 app。
  • 更智能的”Calculate Expression”：支持单位计算与实时汇率、温度、距离换算。
  • 编辑器体验改进：100+ 新图标、富文本预览（日历事件、提醒事项等）、改进的拖拽与变量选择。

2.2 macOS 26 Tahoe 独有亮点

  • 首次支持完整”个人自动化”：Mac 终于像 iPhone 一样可以被事件触发，包括 14 类触发器。
  • Spotlight 深度集成：可直接在 Spotlight（⌘+Space）中运行快捷指令，并将选中的文本作为输入传入。
  • 控制中心 / 菜单栏集成：快捷指令可作为控件添加到控制中心或菜单栏。

2.3 iOS 26 独有亮点

  • Visual Intelligence 动作：通过快捷指令直接打开视觉智能，配合”操作按钮”（Action Button，iPhone 15 Pro 及以后机型）一键启动。
  • 测距仪、提醒事项快速录入等便捷小动作。

2.4 26.4 后续更新

  • 新增 “Set Battery Charge Limit”（电池充电上限）和 “Set Multitasking Mode”（多任务模式）动作。
  • “Open Directions” 支持多站点驾车导航。

5. iPhone 端操作说明（iOS 26）

5.1 找到并打开”快捷指令”app

  • iOS 26 默认预装。如果找不到，下拉主屏幕用 Spotlight 搜索”快捷指令”。
  • 若被卸载，可前往 App Store 免费重装。

5.2 创建你的第一个快捷指令（手动执行型）

  1. 打开”快捷指令”app → 底部点击”快捷指令“标签
  2. 右上角点 + 新建
  3. 点击”添加操作“，搜索关键词（例如 Get Weather）
  4. 按需添加多个动作，按顺序排列
  5. 顶部输入指令名称（例如”早安播报”）→ 选择图标/颜色
  6. 点击右上角”完成“
  7. 长按指令卡片可：分享到主屏幕、添加到 Dock、设置 Siri 触发词

5.3 创建一个”个人自动化”（事件触发型）

  1. 打开”快捷指令”→ 底部”自动化“标签
  2. 点 + → “创建个人自动化”
  3. 选择触发条件（例如”特定时间”→ 早上 7:30，重复”每天”）
  4. 添加要执行的操作
  5. 关闭”运行前询问“（建议）→ 否则每次都需手动确认
  6. 完成

💡 新手提示：第一次设置某些动作（如发送短信、获取位置）会弹出权限请求，请允许。

5.4 利用”操作按钮”（仅 iPhone 15 Pro / 16 / 17 系列）

  • 设置 → “操作按钮” → 选择”快捷指令”
  • 选定任意你创建的快捷指令，长按操作按钮即可一键执行

5.5 从图库（Gallery）下载现成模板

  • 打开 app → 底部”图库“
  • 浏览或搜索（例如”健康”、”工作”、”摄影”），点击后选择”添加快捷指令”
  • 添加后可在”我的快捷指令”中找到并按需修改

6. Mac 端操作说明（macOS 26 Tahoe）

6.1 打开”快捷指令”

  • Launchpad / Spotlight 搜索”Shortcuts”或”快捷指令”
  • 也可在”应用程序”文件夹找到

6.2 创建快捷指令（基本流程同 iPhone）

  1. 顶部菜单栏 → 文件 → 新建快捷指令（或 ⌘N）
  2. 右侧动作面板拖拽动作到中间编辑区
  3. 顶部命名 → 设置图标、颜色
  4. 在右栏”详细信息”中勾选：
    • 在菜单栏中显示
    • 在 Spotlight 中显示（新）
    • 在共享菜单中显示
    • 接受语音输入（Siri）

6.3 创建 Mac 个人自动化（macOS 26 新功能）

  1. 打开”快捷指令”→ 左侧栏”自动化“
  2. 右上角点 +
  3. 选择触发器（见 §3.3 列表，例如”Folder”）
  4. 设置触发参数（例如：监控 ~/Downloads，当文件添加时触发）
  5. 添加动作（例如：移动到指定文件夹、显示通知）
  6. 点击右上角”完成“
  7. 默认会先以”请求确认”模式运行；测试无误后可在自动化详情里切换为”无需确认”

6.4 在 Spotlight 中运行快捷指令

  • 按 ⌘ + Space 打开 Spotlight
  • 直接输入快捷指令的名称 → 回车
  • macOS 26 新特性：如果当前文档中选中了文本，可作为输入传给快捷指令
  • 可在系统设置 → Spotlight 中为常用快捷指令分配”Quick Keys“（例如把”添加新提醒”绑定为 anr）

6.5 添加到控制中心 / 菜单栏

  • 系统设置 → 控制中心 → 编辑控件
  • 在分类中找到”快捷指令”→ 选择 Run Shortcut / Open App / Show “Menu Bar” Collection
  • 可选添加到菜单栏，从此一键调用

7. 最常用最佳实践案例

下面 6 个案例覆盖最高频场景，每个都给出动作清单，可直接照搭。

案例 1：晨间播报（iPhone 自动化）

触发：Time of Day → 工作日 7:30
动作流：

  1. Set Focus → 关闭”勿扰”
  2. Get Current Weather
  3. Get Upcoming Calendar Events
  4. Text（组合文本：”今天 {天气}，你有 {事件} 安排”）
  5. Speak Text（朗读上一步）
  6. Play Music / Play Playlist → 你喜欢的早晨歌单

案例 2：到家自动场景（iPhone 自动化）

触发：When I arrive → 家庭地址
动作流：

  1. Set Focus → 个人
  2. Control Home → 开启客厅灯
  3. Play Music → “晚间放松”歌单
  4. Set Brightness → 50%

案例 3：会议准备一键启动（手动快捷指令）

触发：手动 / Siri
动作流：

  1. Set Focus → 工作
  2. Open App → Zoom（或腾讯会议）
  3. Open App → 备忘录
  4. Create Note → 自动以”YYYY-MM-DD 会议记录”命名

案例 4：URL 清理 + 总结（共享菜单）

用法：在 Safari 看到一篇长文，点分享 → 运行此指令
动作流：

  1. Get URLs from Input
  2. Get Contents of URL
  3. Get Text from Input
  4. Summarize Text（Apple Intelligence 新动作）
  5. Make Note → 保存到 Apple Notes，附原始链接

案例 5：Mac 文件夹监控自动整理（macOS 26 独有）

触发：Folder → 当文件添加到 ~/Downloads
动作流：

  1. Get File Type
  2. IF 条件 → PDF：移动到 ~/Documents/PDFs；图片：移动到 ~/Pictures/Inbox；DMG：14 天后删除
  3. Show Notification → “已整理 1 个文件”

案例 6：低电量节能自动化（iPhone）

触发：Battery Level → 低于 20%
动作流：

  1. Set Low Power Mode → On
  2. Set Brightness → 30%
  3. Turn Bluetooth Off
  4. Show Notification → “已进入省电模式”

案例 7：Apple Intelligence 邮件改写（Mac，新）

用法：在邮件中选中草稿文本，Spotlight 调用此指令
动作流：

  1. Get Selected Text (Spotlight 输入)
  2. Adjust Tone of Text → Professional
  3. Proofread Text
  4. Copy to Clipboard
  5. Show Notification → “已优化并复制”

8. 注意事项与常见问题

8.1 自动化总是要求”确认”

iOS 中”时间”、”专注模式”、”CarPlay 连接”等触发器支持完全自动运行；但”到达地点”等某些触发器默认要求确认。解决方法：

  • 在自动化详情中 → 关闭 “Ask Before Running“（运行前询问）
  • 若仍要求权限，说明某个动作本身需要授权（如发短信），首次执行时允许后续不再询问

8.2 Apple Intelligence 相关限制

  • 需 M1 及以上芯片 的 Mac、iPhone 15 Pro 及以上 机型
  • 支持中文（简体）、英文、日文、韩文等 9 种语言
  • 在中国大陆地区，Apple Intelligence 可能受区域政策影响，需关注 Apple 的最新合规说明
  • “Use Model” 调用 ChatGPT 会经 Apple 中转，无需自带 API Key

8.3 隐私与安全

  • 每个动作首次访问敏感数据（通讯录、位置、相册、健康）都会弹出权限请求
  • 共享给他人的快捷指令在导入时会显示”包含哪些权限请求”，请仔细阅读
  • 不建议运行来源不明的 Run JavaScript / Run Shell Script 类指令

8.4 同步与设备间共享

  • 默认通过 iCloud 在同一 Apple ID 的设备间同步（设置 → Apple ID → iCloud → 快捷指令需开启）
  • 自动化不会同步，每台设备需单独配置
  • 跨平台运行时如果某个动作仅 Mac 支持（如 Folder 触发），iPhone 上会显示”不可用”

8.5 性能与电量

  • 频繁触发的自动化（如”每次解锁屏幕”）会消耗电量
  • 复杂工作流首次运行时延较高，建议拆分成多个小指令
  • Mac 自动化在睡眠状态下不会运行（除非接通电源 + 唤醒因素触发）

8.6 升级后旧快捷指令兼容性

  • iOS 26 / macOS 26 向下兼容绝大部分 iOS 14+ 时代的指令
  • 已弃用动作会标红，可双击查看替代方案
  • 建议升级后逐个打开”我的快捷指令”中的关键指令检查

9. 进阶技巧与扩展

9.1 学会用”变量”

  • 每个动作的输出会成为”魔法变量”（Magic Variable），可被后续动作引用
  • 长按动作中的输入框 → 选择变量 → 即可串联数据流

9.2 用 IF / 选择菜单 / 重复

  • IF：让指令根据条件分支（例如”如果电量 < 20% 则…”）
  • Choose from Menu：让用户运行时弹出多选菜单
  • Repeat with Each：对列表项循环操作（例如批量处理照片）

9.3 与第三方 app 联动

许多优秀第三方 app 提供 Shortcuts Actions，例如：

  • Things 3 / OmniFocus：任务管理
  • Drafts：文本捕获
  • Obsidian（适合知识管理者）：通过 URL Scheme 把文本写入笔记
  • Data Jar / Toolbox Pro：扩展系统能力

9.4 推荐学习资源

  • Apple 官方《快捷指令用户指南》：support.apple.com/guide/shortcuts
  • Matthew Cassinelli 博客：业内最权威的 Shortcuts 评测
  • Reddit r/shortcuts：模板与故障求助社区
  • RoutineHub.co：免费共享指令托管

9.5 给开发者 / 高级用户

  • macOS 上可通过 shortcuts run "ShortcutName" 在终端调用
  • 可与 Automator、AppleScript、Hammerspoon 协作
  • 配合 LaunchAgents 实现更细粒度的定时

  • Apple Support – What’s new in Shortcuts for iOS 26, macOS 26, etc.
  • Apple – Shortcuts User Guide
  • 9to5Mac – iOS 26’s Shortcuts app adds 25+ new actions（2025-12）
  • Macworld – macOS Tahoe Guide（2026-05）
  • MacRumors – macOS Tahoe Hidden Features
  • Matthew Cassinelli – Public Beta Shortcuts deep-dive

结语：快捷指令的精髓在于”先解决一个具体的痛点，再慢慢扩展”。不要试图一次构建一个完美的工作流——从一个 3 步的”晨间播报”开始，体验自动化带来的爽感后，你会自然找到属于自己的高频场景。
