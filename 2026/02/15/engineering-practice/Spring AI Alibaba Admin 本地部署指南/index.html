Spring AI Alibaba Admin 本地部署指南 (MacBook Pro M5 + OrbStack)

1. 环境准备

1.1 硬件与操作系统

  • 设备: MacBook Pro (搭载 M 系列芯片，如 M1/M2/M3/M4/M5)
  • 操作系统: macOS

1.2 核心软件

  • OrbStack: 轻量级、高性能的 Docker Desktop 替代品。用于运行 MySQL、Redis 等中间件容器。
  • Java: JDK 17 或以上版本。
  • Maven: 3.8+ 版本，用于构建 Java 项目。
  • Node.js: 18+ 版本 (推荐 20.x 或 22.x LTS)，用于运行前端前端。
  • pnpm: 推荐使用 pnpm 作为前端包管理器 (npm install -g pnpm)。

1.3 OrbStack 配置建议

OrbStack 默认配置通常能满足需求，但在首次使用时，请确保已启动 OrbStack 并在后台运行。它会自动接管 docker 命令。
